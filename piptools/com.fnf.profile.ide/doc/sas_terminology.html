Fidelity Application Studio Terminology

Table of terms used to describe functionality in the Fidelity Application Studio.

Fidelity Profile Integrates Development Environment The Eclipse plug-in provided by Fidelity Computer Associates Inc. specifically tailored for developers of Fidelity Profile Application functionality. Also known as  Profile IDE.
Fidelity Profile Runtime Environment An instance of the Fidelity Profile product. Also known as Profile RTE.
Fidelity Profile Project A project in the workbench with connectivity properties for communicating with a Profile RTE
