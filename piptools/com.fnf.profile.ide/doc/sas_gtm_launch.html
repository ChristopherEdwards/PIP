GT.M Program Launcher

In this section, you will learn about running M commands in the Studio.

  • M commands can be run from within Fidelity Application Studio through the launch button and menu.

OR
  • Selecting the Run... menu item opens the Run configuration window

  • Select Profile Application and click New
  • Change the Name to clearly describe the action to be executed
  • Click on Browse... and select a Profile project to populate the other fields
  • Make sure that the defaulted information is correct
    • User ID is the operating system user account
    • Password is the operating system password
    • Host is the server name
    • Command to get to Profile is OS specific. Make sure the command and path are correct.
    • Command to get to GTM Prompt is OS specific. Make sure the command is correct.
    • Enter a valid M command
    • You can uncheck the Display login info to cut down on the amount of information displayed in the Console
  • Click Run

  • The results will appear in the Console window.

  • GTM Errors are displayed in red

  • You can quickly re-launch the program by clicking on the Run icon
