Profile Environments View

View and maintain the connection settings to Profile environments.

Features

  • Connection Status List: Displays the connection status for Profile environments.
  • Refresh Button: Updates the Connection Status List.
  • Export Button: Export connection settings for Profile environments. Excludes user ID and password.
  • Import Button: Import connection settings for Profile environments. User ID and password must be set for each environment after import.
  • Add/Update/Delete Connection Buttons: Add/update/delete connection settings for Profile environments.
