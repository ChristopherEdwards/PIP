Column Details View

The Column Details View displays all of the properties of the selected column. Use the edit button to edit the columns properties

Features

  • Column Properties Tab: All of the properties for the selected column.
  • System Map Tab: Lists all programs and line tags where the selected column is read and set. Double click on the Target Name to open PSL source. (Note, not all targets link to PSL source.)
  • Column Documentation Tab: View and edit the documentation for the selected column. Click the update button to save your changes.
