Table Details View

The Table Details View displays all of the properties of the selected table. Use the edit button to edit the tables properties

Features

  • Table Properties Tab: All of the properties for the selected table.
  • Index Tab: Lists all indexes for the selected table. Use Add, Update and Delete buttons to maintain indexes.
  • Foreign Key Tab: Lists all foreign keys for the selected table. Use Add, Update and Delete buttons to maintain foreign keys.
  • Journal Tab: Lists all journals for the selected table. Use Add, Update and Delete buttons to maintain journals.
  • Table Documentation Tab: View and edit the table documentation. Click the update button to save your changes.
