Table List View

The Table List View displays all of the tables along with a the description in the selected environment.

Features

  • Filter Box: Enter text to quickly find tables. Empty out this box to display all tables.
  • Selection Details: Directly to the right of the filter box, this text tells you which environment, table and column you have selected.
  • Table List: A list of all tables in Profile. Select a table to view its properties and columns.
  • Column List: A list of all columns in the selected table. Click on a column to view its details.

Functions

  • Connection Button: Change the Profile environment connection.
  • New Table Button: Create a new table in Profile.
  • New Column Button: Add new columns to the selected table
  • Table Details: Opens the Table Details view if it is not already open.
  • Column Details: Opens the Column Details view if it is not already open.
