What's New in Profile Application Workbench

v 3.3.0
  • Profile Table and Column List View:
    • Connects to Profile Host and displays table list and column list of a table.

  • Table Details View:
    • Displays properties of a selected table and provides an interface for editing details.
    • Allows adding, editing and deleting an Index, Foreign Key or Journal of a table.
    • Displays documentation of a table and also allows updating it.

  • Column Details View:
    • Displays properties of a selected column and provides an interface for editing and updating
    • Allows deleting a selected column with confirmation.
    • Lists System Map of a selected column and displays source codes in a PSL editor when double clicking a map.
    • Displays documentation of a table and also allows for updating.

  • Create a new table and column at host through a wizard

v 3.2.2
  • Project specific keywords
    • Project specific keywords can be generated from MRPC call.
  • Profile Environment View
    • Profile Environment View provides a snap shot of current connection status for each Project settings in your workspace.

  • PSL Help
    • PSL Help release 5.0 is available.
  • FIS Cheatsheet
    • Cheatsheets are available for setting up SQL Explorer and Dayend process.

  • Miscellaneous bug fix
