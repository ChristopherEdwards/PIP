device Property

Class: IO

Description

This property provides the complete specification of the device (with all environment parameters resolved). This is a read-only property. The property receives its value when the open method is called.

Syntax

type string device = IOobject.device

Returns

If the open method has not been called, an empty string is returned.

Otherwise, the complete specification of the device is returned (i.e., directory and filename), with all environment parameters resolved.

When Became Available

v6.4

Example

type IO IO = Class.new("IO")

 

type String EMPTY = IO.device    // empty string

 

set IO.directory = "/home/"

set IO.file = "output.txt"

do IO.open()

 

type String DEVICE = IO.device   // "/home/output.txt"
