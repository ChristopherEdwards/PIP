overlay Method

Class: Record

Description

This method moves user-supplied information from an array into an object. This method is commonly used in conjunction with the default method from the RecordCIF, RecordDEP, and RecordLN classes in MRPCs when creating new customers or accounts from sources other than Profile for Windows.

Syntax

do objectName.overlay(Primitive data)

Parameters

data

An array that is keyed by column names of format "dep.title1".

Example:

data("dep.title1")="John Smith"

data("dep.irn")="3.234"

data ("dep.glsc")="400A"

Returns

Void

When Became Available

Profile v6.0

Example

type RecordDEP dep=Db.getRecord(“DEP”,CID=:CID,1)

if ‘dep.getMode() do {

      type  String data()

      set data(“DEP.TITLE1”)=”John Smith”

      set data(“DEP.IRN”)=”3.234”

      set data(“DEP.GLSC”)=”400A”

do dep.overlay(“data”)

}
