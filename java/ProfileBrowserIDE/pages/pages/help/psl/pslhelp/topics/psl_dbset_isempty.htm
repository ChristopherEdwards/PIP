isEmpty Method

Class: DbSet

Description

This method indicates whether the DbSet (i.e., result set) contains data.

The isEmpty method is no longer required after a Db.select. At one time, this was required to avoid a run-time error if the result set was empty and the cursor advanced using the objectName.next method.

Syntax

objectName.isEmpty()

Parameters

None

Returns

1 if the result set is empty.
0 if the result set contains data.

When Became Available

v6.4

Example

type String BRCITY,BRSTATE,BRZIP,DESC

type DbSet ds = Db.selectDbSet(“UTBLBRCD”)

if ds.isEmpty() quit

while ds.next() do {

  type RecordUTBLBRCD utblbrcd = ds.getRecord(“UTBLBRCD”)

  set DESC = utblbrcd.desc

  set BRCITY = utblbrcd.brcity

  set BRSTATE = utblbrcd.brstate

  set BRZIP = utblbrcd.brzip

  }
