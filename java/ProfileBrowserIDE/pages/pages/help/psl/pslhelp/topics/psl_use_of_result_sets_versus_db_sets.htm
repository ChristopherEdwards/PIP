Use of Result Sets Versus Db Sets

Result Sets

If you plan to simply retrieve data, use resultsets followed by Db.getRecord. As seen below, this produces less code than a DbSet and does not instantiate an object.

Example

type ResultSet phldrs = Db.select("COMP","PHLD","CID=:CID")

while phldrs.next()  do {

set COMP = phldrs.getCol("COMP")

}

Db Sets

If you plan to instantiate records, use db sets.

Example

type DbSet phldrs = Db.selectDbSet("PHLD","CID=:CID")

while phldrs.next()  do {

type String COMP

type RecordPHLD phld = phldrs.getRecord("PHLD")

set COMP = phld.comp

}
