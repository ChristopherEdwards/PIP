journal Property

Class: Column

Description

This property indicates whether the filer should create a journal entry to track a change made to the column. This property requires that you invoke the setAuditFlag method prior to the column change. Because SQL generates object code, the default for this flag is to generate a journal of the changes.

Syntax

table.column.journal

Parameters

1 to enable journaling (default)

0 to disable journaling

Returns

Void

When Became Available

v6.0

Example

type RecordDEP dep = Db.getRecord(“DEP”,”CID=:CID”)

do dep.setAuditFlag(1)

set dep.bal.journal = 0

 

* Variable CID must be defined prior to the getRecord statement.
