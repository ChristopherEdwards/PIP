PSL Programming Standards

The Java Coding Standards document describes general standards and conventions related to object-oriented design and programming. The Profile Programming Standards online Help describes the standards and guidelines for programming Profile applications in M programming language and the Profile Scripting Language (PSL).

Per the Fidelity Information Services Software Prcess (FISSP), code inspections must be performed on all PSL code. A checklist of the PSL code inspection criteria appears on the FISSP intranet site.
