charsetElemType Property

Class: IO

Description

This property specifies the element type that is used by IO.open() to derive the charsetName that is used for character translation. It is combined with the value of the charsetElemName property, the subroutine name, and the instance variable name. See IO.open() for details.

 

Only the following case-sensitive values are valid for this property:

  • "Routine"

  • "Report"

Syntax

type String IOObject.charsetElemType = "Report"

Returns

A String

When Became Available

Profile V7.0

Example

type IO IOD = Class.new("IO")

 

type String EMPTY = IOD.charsetName // empty string

 

set IOD.charsetElemName = "MyReport"

set IOD.charsetElemType = "Report"

 

set IOD.directory = "/home/"

set IOD.file = "output.txt"

do IOD.open()

 

type String CHARSET = IOD.charsetName // may have been set by open()
