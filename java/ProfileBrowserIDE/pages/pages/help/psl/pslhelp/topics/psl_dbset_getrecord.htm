getRecord Method

Class: DbSet

Description

This method retrieves individual records from a DbSet object. Use the Db.getRecord method when instantiating a single Record<Class> object versus a set of Record<Class> objects. To retrieve multiple records simultaneously, use the Db.select method. To define a new object in create mode, use the Class.new method.

This method supports lowercase variables, object.property, and system variables within the host variable syntax. PSL maps these variables and object references into uppercase host variables.

When the Db.getRecord method is used to instantiate a literal scope variable, the access keys must be either string literals or literal variables.

Record<class> Properties

As of Profile v6.4, Record<class> properties can be referenced through the DbSet.getRecord method directly, without having to explicitly instantiate a Record<class> object. PSL implicitly instantiates the Record<class> object with the same scope as the DbSet object. For example, the following passes the RecordDEP properties:

type RecordDbSet ds = Db.seleectDbSet("DEP","BAL>100")

while ds.next() do {

    set x = ds.getRecord() .cid

    set y = ds.getRecord() .bal

}

Declaration of DbSet Object

type DbSet Ds = Db.selectDbSet(...)

Syntax

type RecordTABLE objectName = ds.getRecord(String tableName)

Parameters

tableName

A valid database table name used in the Db.selectDbSet method. This parameter is not required in the subroutine where the DbSet object was instantiated. It defaults to the table used to create the DbSet. However, this parameter is required in the first reference to the object in a called procedure if a DbSet object is passed into the procedure. For clarity, it is recommended that you include this parameter in all calls.

Returns

An object of type Recordtable (e.g., RecordDEP, RecordLN) containing a single record based on the WHERE clause and ORDER BY clause in the specified Db.selectDbSet call.

When Became Available

v6.4

Example

type DbSet ds = Db.selectDbSet(“UTBLBRCD”)

type String BRCITY,BRSTATE,BRZIP,DESC

while ds.next() do {

  type RecordUTBLBRCD utblbrcd = ds.getRecord(“UTBLBRCD”)

  set DESC    = utblbrcd.desc

  set BRCITY  = utblbrcd.brcity

  set BRSTATE = utblbrcd.brstate

  set BRZIP   = utblbrcd.brzip

  }
