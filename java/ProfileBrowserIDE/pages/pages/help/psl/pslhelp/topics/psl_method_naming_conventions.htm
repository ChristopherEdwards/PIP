Method Naming Conventions

Use the following syntax when naming methods:

Class.method()

where

  • Class - The name of the class to which the method belongs. Capitalize the first letter of each word in the class name.

Example

if ‘Class.isClass("RecordDEP") quit

else  type ResultSet rs = Db.select(“CID,BAL,LNM”,”DEP”)

  • method - A lowercase letter for the first word in the method, followed by initial caps for each of the following words in the method name.

Example

type RecordDEP dep=Db.getRecord(“DEP”,”CID=:CID”)

do dep.save()
