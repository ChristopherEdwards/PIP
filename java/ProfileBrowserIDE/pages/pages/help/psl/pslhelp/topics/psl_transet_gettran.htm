getTran Method

Class: TranSet

Description

This method retrieves a transaction from a transaction set object.

Syntax

transactionObject.getTran(transactionSeq)

Parameters

transactionSeq

The number identifying the transaction to retrieve.

Returns

The object name.

When Became Available

Profile v6.0

Example

type TranSet ts=Class.new(“TranSet”)

set TSEQ = ””

for set TSSEQ=Db.nextKey(“TTX”,”:TSEQ”) quit:TSEQ = ”” do {

       type RecordTTX ttx = ts.getTran(x(TSEQ))

       set RTY = ttx.rty

 

       if rejmet>0 set STAT = ts.status

       // Rejected transactions

 

       else set STAT = $P(ts.status,”|”,TSEQ)

       }
