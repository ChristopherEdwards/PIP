toArray Method

This method has been deprecated and should not be used in Profile versions v6.4 and above.

Class: Record

Description

This method copies a PSL object into an M array. Use this method to ensure backward compatibility with existing visible M data structures. This method copies the object from its hidden structure to the M structures previously used to store database records. In addition, this method defines the primary access keys as local variables. If the setAuditFlag is set when the object is updated, this method creates the UX structure used in non-PSL code. The UX structure contains the old value and new value for each changed column.

Syntax

objectName.toArray(Literal MArrayReference,Literal keys)

Parameters

MArrayReference

The reference to a local M array.

keys

A comma-separated field that provides a mechanism to have the keys of a record being copied, returned in variables as well as within the object.

Returns

Void

When Became Available

Profile v6.4

Example

type RecordDEP dep = Db.getRecord("DEP","CID")

 

set dep.boo = BRANCH

 

do dep.toArray("%A(CID") // into %A(CID,n)

do dep.toArray("A") // into A(n)

do dep.toArray("ABC(""XYZ""") // into ABC("XYZ",n)
