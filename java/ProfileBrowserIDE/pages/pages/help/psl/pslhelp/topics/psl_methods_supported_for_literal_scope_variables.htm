Methods Supported for Literal Scope Variables

Literal scope variables can be typed as Record<Class> or ResultSet objects. This provides a mechanism for including database records or column values in the generated code. The following methods are supported for literal scope variables:

  • Db.getRecord

  • Db.isDefined

  • Db.select

  • ResultSet.next

  • ResultSet.getCol

  • ResultSet.getRow

Example

The following PSL code:

tag(array)

type literal ResultSet RS = Db.select("BRCD,CCDEF,DESC","UTBLBRCD")

#while RS.next()

set array(RS.getCol(1)) = $P(RS.getRow("|"),"|",2,3)

#end

quit

will generate the following M code:

tag(array) ;

;

S array("0") = "0|Back Office"

S array("1") = "1|Br 1"

S array("2") = "2|Front-End Branch 2"

S array("3") = "2|Front-End Branch 3"

When Standard is Enforced: Profile v7.0
