Use of #WHILE Compiler Command

The #WHILE command, used in conjunction with the following, provides a mechanism to generate code based on the values of database tables:

  • A literal scope ResultSet object

  • ResultSet.next method

  • ResultSet.getRow or ResultSet.getCol method

This feature can be used to load static tables directly into programs as arrays and/or to generate code logic using database column values as conditional tests and code expressions or functions.
