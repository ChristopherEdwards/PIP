recordSize Property

Class: IO

Description

This property defines the length of the record within the device to be accessed. The default is 512. The maximum record size depends on the operating system.

Syntax

objectName.recordSize=record_size

Returns

A number containing the recordSize that will be applied in the next call to IO.open.

When Became Available

v6.4

Example

type IO io = Class.new(“IO”)

set io.directory = ”/unix_directory/spool”

set io.fileName = ”TEST.txt”

set io.openParams = ”NEWV/WRITE”

set io.recordSize = 700

set io.timeout = 5

 

do io.open()

type String TEXT

set TEXT = ”This is a file created using IO class methods.”

do io.write(TEXT)

do io.close()

 

* The value of the directory property must be a valid directory path.
