Use of #IF/#ELSE and Literal Scope Variables

The most effective way to eliminate database IO is to eliminate code that contains database IO. In the following example, the ResultSet in the #IF block is eliminated from the generated code if the #IF condition is false.

Example

#IF CUVAR.%MCP

type ResultSet rs = Db.select("CRCD","UTBLCRCD")

while rs.next() set crcd(rs.getRow()) = ""

#END

Literal scope variables replace the PSL reference with the actual value of the variable at compile time. This can eliminate run-time database IO and potentially eliminate run-time indirection and execution as well.

Example

type String %LIBS = CUVAR.%LIBS // Will set to %LIBS="SYSDEV"

type Literal String routine = CUVAR.BWAPGM

#IF routine = "" set return = $$XYZ(CID)

#IF routine ’= "" xecute "set return = $$"_routine_"(CID)"
