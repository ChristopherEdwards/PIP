data Method

This method has been deprecated and should not be used in Profile versions v6.4 and above. It combines information from Object.exists() and Object.order(). Use those methods instead.

Class: Object

Syntax

Object.data()

Parameters

None

 

Returns

0

If Object.exists() = 0, and Object("").order() returns an empty value.

1

If Object.exists() = 1, and Object("").order() returns an empty value.

10

If Object.exists() = 0, and Object("").order() returns a non-empty value.

11

If Object.exists() = 1, and Object("").order() returns a non-empty value.

When Became Available

 

Example

// declare object that is also used as a multi-level array

type Number XX,DATA

type String XX(),XX(,)

 

set DATA = XX.data()            // 0

set DATA = XX(17).data()        // 0

set DATA = XX(17,33).data()     // 0

set XX = 24, DATA = XX.data()   // 1

set XX(17,33) = "A String"

set DATA = XX.data()            // 11

set DATA = XX(17).data()        // 10

set DATA = XX(17,33).data()     // 1
