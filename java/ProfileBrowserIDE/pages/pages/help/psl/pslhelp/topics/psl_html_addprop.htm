addProp Method

Class: HTML

Description

This method is used with the HTML.addString method. In situations where data being returned is not retrieved from a ResultSet or Db.getOneRow method, use the addProp method to add single entries to the document.

Syntax

htmlObject.addProp(String property, String value)

Parameters

property

The individual column name that is being added to the XML document.

value

The value of the column being added to the XML document.

Returns

An updated XML formatted string updated by one individual column and data value.

When Became Available

v6.4

Example

type HTML xx = Class.new(“HTML”)

set return = xx.addString(“ERROR”,0)

set return = xx.addProp(“DESCRIPTION”,RM)

set return = xx.addString(“ERROR”,1)

____________________________________________________________

This will return the following information in the XML document, assuming RM has the value that follows:

<DESCRIPTION> <ERROR>

<DESCRIPTION> This account is closed</DESCRIPTION>

</ERROR>
