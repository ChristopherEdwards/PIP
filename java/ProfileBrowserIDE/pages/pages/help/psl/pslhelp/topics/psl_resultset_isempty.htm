isEmpty Method

Class: ResultSet

Description

This method indicates whether the results set contains data.

The isEmpty method is no longer required after a Db.select. At one time, this was required to avoid a run-time error if the result set was empty and the cursor advanced using the objectName.next method.

Syntax

objectName.isEmpty()

Parameters

None

Returns

1 if the result set is empty.
0 if the result set contains data.

When Became Available

Profile v6.0

Example

type ResultSet rs = Db.select("ACN,NAM,TAXID","CIF")

if rs.isEmpty() quit

while rs.next()  do {

       set ACN   = rs.getCol(1)

       set NAM   = rs.getCol(2)

       set TAXID = rs.getCol(3)

       }
