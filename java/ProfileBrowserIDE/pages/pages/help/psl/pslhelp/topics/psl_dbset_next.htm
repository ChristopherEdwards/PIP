next Method

Class: DbSet

Description

This method positions the cursor at the next row of the DbSet (i.e., result set). When the result set is created, the cursor is positioned before the first row. Use this method to position the cursor on the first row, and then on subsequent rows.

Syntax

objectName.next()

Parameters

None

Returns

1 if more rows exist in the result set.
0 if the cursor is at the end of the result set, or if there are no rows returned from a Db.selectDbSet.

When Became Available

Profile v7.0

Example

type String BRCITY,BRSTATE,BRZIP,DESC

type DbSet ds=Db.selectDbSet(“UTBLBRCD”)

if ds.isEmpty() quit

while ds.next() do {

  type RecordUTBLBRCD utblbrcd = ds.getRecord("UTBLBRCD")

  set DESC    = utblbrcd.desc

  set BRCITY  = utblbrcd.brcity

  set BRSTATE = utblbrcd.brstate

  set BRZIP   = utblbrcd.brzip

  }
