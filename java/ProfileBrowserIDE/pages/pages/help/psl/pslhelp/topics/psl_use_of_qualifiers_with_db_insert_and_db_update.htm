Use of Qualifiers with Db.insert and Db.update

A standard set of qualifiers is processed when the filers are executed. This standard set includes:

  • NOVALDD

  • NOVALRI

  • NOVALREQ

  • NOFKCHK=1

In addition, a number of filer qualifiers can be associated with the Db.insert and Db.update methods to eliminate processing within various sections of the filer if it is not required. These qualifiers include the following:

  • NOVALDD

  • NOVALRI

  • NOVALREQ
